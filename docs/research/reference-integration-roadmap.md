# Reference Integration Roadmap

This project uses external repositories only through audited, license-aware integration.

## Directly Adapted
- `blacktwist/social-media-skills`: MIT. Adapted as generated skill metadata and workspace skill cards.

## Conceptual Reference Only
- `langchain-ai/social-media-agent`: MIT, but API/SaaS-oriented. Recreated locally as prompt-packet workflow, approval gate, and review loop.
- `microsoft/TinyTroupe`: MIT. Recreated as deterministic persona simulation and advisory buyer-persona reports.
- `idiap/sdialog`: MIT. Recreated as persona reaction/report patterns, not imported runtime code.

## Restricted / Not Vendored
- AGPL repositories stay reference-only unless the whole distribution strategy explicitly accepts AGPL obligations.
- No-license repositories are reference-only.
- High ToS-risk automation stays disabled and outside core.

## Current Integration Status
- Skill taxonomy: integrated.
- Human approval workflow: integrated.
- Persona simulation: integrated.
- Local workflow runner: integrated.
- Publishing package: integrated as manual/official path only.
- Image generation: pending optional ComfyUI module.
- Browser-assisted trend evidence: planned as human-approved public source collection.
